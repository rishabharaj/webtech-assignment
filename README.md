# Simple Portfolio Website

A minimal and clean portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on all devices
- **Smooth Scrolling**: Enhanced navigation experience
- **Contact Form**: Integrated with Web3Forms for email submissions
- **Modern UI**: Dark theme with accent colors
- **Animations**: Scroll-based reveal animations and smooth transitions

## Structure

```
simple-web/
├── index.html      # Main HTML file
├── style.css       # Styling and layout
├── script.js       # JavaScript functionality
└── README.md       # Documentation
```

## Sections

1. **Home/Hero**: Introduction and call-to-action buttons
2. **About**: Personal information, stats, and services
3. **Skills**: Technical skills organized by categories
4. **Projects**: Featured projects with links
5. **Contact**: Contact information and form

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- Web3Forms API (for contact form)

## How to Use

1. Open `index.html` in any web browser
2. Navigate through sections using the navigation menu
3. Modify content in `index.html` to personalize
4. Customize colors and styles in `style.css`
5. Adjust functionality in `script.js` as needed

## Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --bg-primary: #0a0a0a;
    --bg-secondary: #1a1a1a;
    --text-primary: #ffffff;
    --text-secondary: #a0a0a0;
    --accent-color: #ff6b6b;
    --border-color: #333;
}
```

### Content
Update the text and links in `index.html` to match your information.

### Contact Form
Replace the Web3Forms access key in `script.js` with your own key if needed.
